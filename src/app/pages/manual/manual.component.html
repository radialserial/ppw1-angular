Introdução

World Wide Web

  Originalmente envisionada por Tim Berners-Lee enquanto trabalhava na CERN nos anos 80, a rede de computadores mundial é um sistema de transmissão e gerência de documentos de texto, representados primáriamente por três tecnologias: HTML, CSS, e JavaScript.

Conforme a web evoluiu, porém, seu escopo transcendeu seu objetivo inicial, e se tornou algo completamente diferente. É difícil observar um website hoje em dia e interpretálo como um mero documento de texto.

De simples manuais em texto comum a vídeos, programas, e até mesmo videogames, as páginas da web contemporânea são capazes das mais diversas utilidades, dependendo da necessidade do usuário.

Funcionamento

A web, resumidamente, funciona em um sistema de clientes e servidores, onde clientes acessam servidores para receber páginas codificadas que são entçao interpretadas pelo navegador utilizado.

Tais páginas tem sua estrutura divida em três partes: conteúdo, estilo, e funcionalidade - cada uma dessas partes é representada por uma das linguages utilizadas em desenvolvimento web.

HTML é a linguagem primáriamente utilizada para estruturar documentos, embora nem sempre tenha exercido apenas essa função. Hoje em dia, também tem como papel descrever semântica dentro das páginas, facilitando a interação de mecanismos automatizados com websites modernos.

CSS é utilizado para introdução de estilos avançados e semi-interativos em páginas da web, sendo capaz de gerenciar as cores, posicionamento, e até mesmo animações de elemento de uma página da web.

JavaScript a principal linguagem de programação utilizada em webdev, seu papel o de adicionar funcionalidade avançada a websites, sendo capaz de gerenciar ambos back e front-end.

HTML

Básicos

A Linguagem de Marcação de Hiper-Texto é a tecnologia principal mais utilizada em webdev. É a base de todo o desenvolvimento das páginas online e offline, e não tão complicada quando são compreendidos os básicos absolutos.

HTML funciona sobre um sistema de tags, representadas por uma sintaxe específica. Um exemplo de HTML: 

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>

    </body>
    </html>

Cada tag em HTMl tem uma função diferente:

  • html: define os confins do documento HTML.
  • head: usada para definir bloco onde são inseridos os metadados da página.
  • title: define o título da página.
  • meta: usada para definir metadados individuais como o ícone, palavras-chave, descrição da página, dentre outros.
  • body: onde é colocado o conteúdo principal a ser exibido na página.

Como HTML é uma tecnologia sempre em desenvolvimento, tags entram em desuso constantemente, tal como tags novas são criadas para extender a funcionalidade da linguagem. Por exemplo, as tags center e font não são mais suportadas em HTML5, a versão mais recente de HTMl.

CSS

Básicos

A linguagem de Folhas de Estilo em Cascata é um sistema de introduzir estilos avançados em HTML de forma hierárquica. Um exemplo: 

    footer {{"{"}}
      position: fixed;
      left: 0;
      bottom: 0;
      width: 100%;
      text-align: center;
      background-color: #282A36;
    {{"}"}}

Esse bloco controla o estilo da tag footer, que age como o rodapé da página. Cada um dos atributos controla uma característica diferente do rodapé; width controla a largura, background-color controla a cor do fundo, e assim por diante.

Frameworks

Básicos

Frameworks são sistemas feitos para facilitar o desenvolvimento de bases de código complexas. Elas automatizam e facilitam a geração de código repetitivo e maçante.

Frameworks geralmente são baseados em linguagens de programação específicas, como Spring (baseado em Java) ou Django (baseado em Python), dentre outras.

A maioria dos frameworks são baseados em javascript. Alguns dos mais populares são Node, React, e Angular.
